# SVG Shrinker

Another GUI for SVGO. Made with Electron

## How to use

Drag your SVG in the window and it will optimized and saved as yourfilename.min.svg

Easy, bitch!

## Installation
Get the repo
```shell
git clone https://lab.clickpress.de/stefan.sl/svg-shrinker.git
```
Install dependencies
```shell
$ cd svg-shrinker
$ npm install
```
Generate your MacOS package
```shell
npm run pack-mac
```
Generate your Linux package
```shell
npm run pack-linux
```
